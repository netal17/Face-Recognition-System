# Face Recognition Attendance System

A web-based Face Recognition Attendance System built using **Python**, **Flask**, **OpenCV**, **face_recognition**, and **SQLAlchemy**. The application automatically recognizes registered users through a webcam, records attendance in real time, and provides an interactive dashboard for attendance analytics and member management.

---

## Features

- Real-time face detection and recognition
- Automatic attendance marking
- Member registration and management
- Dashboard with attendance statistics
- Attendance history and logs
- Face image storage in database
- Interactive charts using Plotly
- Flask-based web interface
- SQLAlchemy database integration

## Workflow

1. Register users with their facial images.
2. Store face encodings in the database.
3. Capture live video from the webcam.
4. Detect and recognize faces.
5. Match detected faces with stored encodings.
6. Automatically record attendance.
7. Display attendance records and analytics on the dashboard.

## Installation

Clone the repository

```bash
git clone https://github.com/netal17/Face-Recognition-Attendance-System.git
```

Move into the project

```bash
cd Face-Recognition-Attendance-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
